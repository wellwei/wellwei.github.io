# 附：
## 关于Vjudge平台提交代码时需要某平台Cookie的解决方案

### 以洛谷平台为例（其他平台操作方法一样）
![QQ_1726756350519](https://github.com/user-attachments/assets/588c3912-c4a7-4495-a7c4-76fbcf3815d5)


- 进入[[洛谷官网](https://github.com/wellwei/wellwei.github.io/issues/luogu.com.cn)](luogu.com.cn)注册自己的账号
- 按键盘 F12 或 右击网页点击*检查* 进入Web调试页面
- 点击应用程序(applications)，若无该按钮点击+号添加
![QQ_1726756678402](https://github.com/user-attachments/assets/84336374-aae4-4386-aee9-2c2d55bca1e8)

- 选择Cookie并将其_uid和_client_id字段填入Vjudge需求框中即可
![QQ_1726756770210](https://github.com/user-attachments/assets/29b09dab-6ed5-402d-9f42-bedd5f06aafc)

- 若失败可检查网络或稍后重试
